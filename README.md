### Installation

- Just, Clone this repository -
 
  ```
  git clone --depth=1 https://github.com/htr-tech/zphisher.git
  ```


  ```
  $ cd zphisher
  $ bash zphisher.sh
  ```

- On first launch, It'll install the dependencies and that's it. ***Zphisher*** is installed.
